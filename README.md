# Computational Brain Modeling

## Overview

This project explores computational modeling of neural dynamics using mathematical and simulation-based approaches.

The objective is to implement and analyze models inspired by biological neural systems to study stability, system behavior and dynamical properties.

## Methodology

The project includes:

- Mathematical formulation of the neural model
- Numerical simulation of system dynamics
- Parameter analysis and stability exploration
- Visualization of model behavior over time

## Skills Demonstrated

- Computational Modeling
- Dynamical Systems
- Mathematical Analysis
- Python for Scientific Computing
- Simulation and Data Visualization

## Repository Structure

- `brain_modeling.ipynb`: Full notebook implementation
- `requirements.txt`: Dependencies
